# AI 视频创作教程与作品集

本仓库用于存放使用 AI 进行视频创作的教程文档、项目中间产物和最终成品。

采用「项目为主 + 公共资源为辅」的混合结构，既保证每个作品的完整链路，也方便沉淀可复用的提示词、工作流和经验。

## 目录结构

```
ai-video-creation/
├── README.md              → 仓库总览
│
├── shared/                → 跨项目复用的公共资源
│   ├── prompts/           → 通用提示词（画面、镜头、风格等）
│   ├── workflows/         → 工作流文档（Sora / Runway / Kling / ComfyUI 等）
│   ├── templates/         → 模板文件（封面、字幕条、视频模板）
│   └── tools/             → 辅助脚本、配置文件、批量处理工具
│
├── tutorials/             → 通用教程、经验总结、踩坑记录
│   └── ...
│
└── projects/              → 按项目组织的完整创作链路
    └── YYYY-MM-project-name/
        ├── README.md      → 项目说明、目标、使用工具、成品链接
        ├── concept.md     → 创意概念、分镜脚本、参考图
        ├── prompts/       → 项目专属提示词
        ├── storyboards/   → 分镜、草图、参考图
        ├── assets/        → 素材（图片、音频、视频片段等）
        ├── drafts/        → 草稿、中间产物、迭代版本
        └── final/         → 最终成片、封面、发布文案
```

## 项目命名规范

每个项目目录统一使用：`YYYY-MM-简短英文标题`

示例：
- `2026-06-sunset-city`
- `2026-07-forest-spirit`

## 使用建议

1. 开始新项目时，在 `projects/` 下按规范创建目录。
2. 可复用的提示词、工作流、模板优先放到 `shared/`。
3. 每个项目结束后，确保 `final/` 中包含最终成片和发布相关文件。
4. 通用经验和教程整理到 `tutorials/`，方便后续查阅和分享。
