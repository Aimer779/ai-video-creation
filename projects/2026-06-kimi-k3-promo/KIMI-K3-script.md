# Kimi K3 宣传片《三月纪》制作教程

> **核心概念**：「3」不是被发明的数字，而是月亮写在天上的签名，是人类探索文明中反复出现的轨迹。

---

## 一、项目基本信息

| 参数 | 数值 |
|------|------|
| 分辨率 | 720 x 720 px（正方形） |
| 帧率 | 24 FPS |
| 总时长 | 约 20 秒 |
| 画面风格 | 东方古籍美学 + 太空探索影像 |
| 核心色调 | 暖宣纸黄 → 深空靛蓝（单镜头内完成色温跃迁） |
| 核心符号 | 数字「3」与新月负形的同构 |

## 二、完整时间线拆解

### 【第一阶段】古籍中的三（0:00 – 0:10）

**叙事目标**：建立「3」作为文明密码的古老身份。不是展示 AI，而是展示人类在长文本、探索、连接中早已写下「3」。

**节奏**：每张素材 **12 帧（0.5 秒）**，交叉淡化 **5 帧（约 0.2 秒）**。整体带有缓慢推进缩放与极微顺时针旋转（0° → 1.5°）。

**节奏**：每张素材 **14 帧（约 0.6 秒）**，稍缓于第一阶段，让观众有时间识别「3」的形态。继续交叉淡化。

### 【第三阶段】三月升空（0:16 – 0:19）

**叙事目标**：静物活化。从「自然界有 3」跃迁到「3 开始运动、连接、涌现」。这是全片唯一的大动态段落，替代 Anthropic 的「蝴蝶飞散」，Kimi 使用**「光点升空」**——从地面古籍飞向深空星图。

| 时间 | 画面内容 | 技术要点 |
|------|----------|---------|
| 0:16.250 | 新月照片微微发光，银辉溢出边缘 | 使用 **Glow + Radial Blur**，颜色 #E8F4F8 |
| 0:16.750 | 月光如液态水银流淌，画出一条发光的「3」形轨迹 | **Particular 或手动路径动画**，轨迹带拖尾 |
| 0:17.250 | 轨迹分裂成无数光点，向上飞升 | 粒子发射器沿 3 形路径分布，向 Y 轴负方向运动 |
| 0:17.750 | 光点在深空中连成神经网络/星图，节点闪烁 | 粒子连线（Plexus 或手动 Shape Layer 路径）|
| 0:18.250 | 网络收缩，凝聚成一弯明亮的新月，悬于深空 | 粒子汇聚（Physics → Attract），最终形态与品牌 Logo 新月重合 |
| 0:18.750 | 新月脉动一次，光芒轻微扩散 | Scale 100% → 103% → 100%，Glow 强度同步脉动 |

**技术特征**：
- 光点需分 **3-4 个景深层次**：
  - 远景：小、蓝、高模糊（Gaussian Blur 8px）
  - 中景：中、白、微模糊
  - 近景：大、暖白、清晰、带运动模糊
- 颜色过渡：**暖银（#C0C0C0）→ 冷蓝（#4A90E2）→ 月白（#E8F4F8）**
- 背景从**宣纸黄**渐变为**深空蓝（#0A0E27）**，这是全片最大的色温跃迁

### 【第四阶段】品牌卡（0:19 – 0:20.4）

| 时间 | 画面内容 |
|------|----------|
| 0:19.000 | 推镜头进入深空背景，新月与品牌图标重合 |
| 0:19.250 | **Kimi K3** 文字淡入居中，新月悬于文字上方 |
| 0:20.000 | 底部小字 **Moonshot AI** 淡入 |
| 0:20.250 | 大光晕轻微脉动，视频结束 |

**设计规范**：
- **背景**：深空蓝（#0A0E27）到纯黑径向渐变，叠加微弱星图纹理（不透明度 8%）
- **「Kimi」字体**：英文使用 **Canela** 或 **Freight Display**（现代衬线，有雕刻感）；中文使用 **思源宋体 Heavy** 或 **方正清刻本悦宋**（保留雕版刀刻特征）
- **「K3」设计**：「3」的轮廓可设计为**新月的负形**——月牙的弧度与 3 的上半圆共用一条曲线；或让「K」的最后一笔自然延伸为 3 的弧线
- **新月图标**：发出柔和冷光（#E8F4F8），Glow 半径 20-30px，轻微呼吸动画（Scale 98% → 102%，4 秒循环）
- **字间距**：稍微收紧（Tracking -20），营造古籍标题的紧凑感

## 三、核心技术拆解

### 1. 交叉淡入淡出（Crossfade Dissolve）

与 Anthropic 原片一致，这是节奏的基石。

```
图层A: 不透明度 100% → 0%（最后5帧）
图层B: 不透明度 0% → 100%（开头5帧）
重叠可见时间：3-4 帧
缓动：线性（Linear）
```

### 2. Ken Burns 呼吸感

每张静态素材必须「活」着：

```
缩放: 100% → 108%（持续整个素材时长）
旋转: 0° → 1.5°（顺时针）
位置: 随机 ±8px 偏移（避免所有素材中心对齐）
```

### 3. 第三阶段粒子系统（光点升空）

**方案 A：Trapcode Particular（推荐）**

```
发射器类型：Layer（使用 3 形路径图层）
粒子数量：150-200
速度：Y轴 -80（向上），X轴随机 ±30
空气阻力：2.0
粒子生命周期：2-3 秒
粒子类型：Glow Sphere
颜色映射：从 #C0C0C0（暖银）到 #4A90E2（冷蓝）
```

**方案 B：After Effects 内置 CC Particle World（无插件）**

```
发射器位置：沿 3 形 Mask Path 关键帧移动
物理属性：重力设为负值（向上飘）
添加调整图层：Glow + Gaussian Blur（远景层）
```

**景深模拟（无插件方案）**：
- 复制粒子层 3 份，分别命名为「远/中/近」
- 远景：Scale 40%，Gaussian Blur 12px，不透明度 60%
- 中景：Scale 80%，Gaussian Blur 3px
- 近景：Scale 120%，无模糊，添加 Motion Blur

### 4. 新月脉动

```
新月图层：
  Scale: 100% → 103% → 100%（缓入缓出，2秒循环）
  Glow Intensity: 100% → 130% → 100%（与 Scale 同步）
```

## 四、质感协议：从宣纸到深空

这是 Kimi K3 与 Anthropic Fable 5 最大的差异化所在。

### 1. 第一阶段调色（古籍段）

```
色温: +25（偏暖黄）
色调: 轻微偏向红 +5
对比度: -15（柔化古籍质感）
饱和度: -8（做旧感）
高光: -20（压缩过曝）
阴影: +15（提亮暗部，保留宣纸纤维细节）
```

**纹理叠加**：
- 在最上层叠加 **宣纸纹理**（混合模式 Soft Light，不透明度 12%）
- 叠加 **竹简/毛边纸边缘**（Multiply 模式，不透明度 8%，仅左右边缘）
- 添加 **细微灰尘划痕**（Overlay 模式，不透明度 5%）

### 2. 第二阶段调色（万物段）

```
色温: +10（保持微暖，但向中性过渡）
对比度: 0（自然物保留真实对比）
清晰度: +10（突出 3 形边缘）
```

### 3. 第三阶段调色（升空段）

```
色温: -40（大幅偏冷）
色调: 偏向青 +15
对比度: +20（深空需要深邃感）
高光: +10（光点更亮）
阴影: -30（压暗背景，突出粒子）
```

**背景过渡**：
- 使用 **4-Color Gradient** 或 **Ramp** 从 #E8DCC8（宣纸黄）渐变到 #0A0E27（深空蓝）
- 过渡时机：从 0:16 开始，持续 3 秒完成

### 4. 全局颗粒

```
添加调整图层：Noise & Grain → Add Grain
预设：Kodak 5247 或类似中等颗粒
数量：3-5%
大小：0.4-0.6
混合模式：Overlay，不透明度 15%
```

## 六、分步制作教程（After Effects）

### 步骤 1：项目设置

```
合成设置：
  - 预设：自定义
  - 宽度：720 px
  - 高度：720 px
  - 帧率：24 fps
  - 持续时间：20 秒
  - 背景色：#E8DCC8（宣纸黄）
```

### 步骤 2：导入与分类

```
项目面板文件夹结构：
  /01_古籍素材
  /02_万物素材（含3）
  /03_升空粒子
  /04_品牌元素
  /05_纹理与调色
  /06_音频（如有）
```

所有素材预先在 Photoshop 中统一为 **720x720 或更大正方形**，分辨率 150dpi 即可（模拟扫描质感）。

### 步骤 3：制作第一阶段（古籍蒙太奇）

```
1. 新建合成 "Part1_Ancient"
2. 按时间线顺序拖入 10 张古籍素材
3. 每张素材时长：12 帧（0.5 秒）
4. 素材之间重叠 5 帧
5. 选中所有图层，按 Alt + [ / Alt + ] 精确裁剪入出点
6. 为每张素材添加关键帧：
     Scale: 100% → 108%
     Rotation: 0° → 1.5°
     Position: 随机偏移 ±8px
7. 全选图层 → 右键 → 关键帧辅助 → 序列图层（重叠 5 帧，过渡：溶解前景图层）
8. 在最上层添加调整图层 "Grade_Part1"，应用：
     Curves（压高光提阴影）
     Color Balance（红 +5，黄 +10）
9. 添加宣纸纹理图层，混合模式 Soft Light，不透明度 12%
```

### 步骤 4：制作第二阶段（万物中的三）

```
1. 新建合成 "Part2_Nature"
2. 拖入 12 张「本身就是 3」的素材
3. 每张时长：14 帧（约 0.6 秒），重叠 5 帧
4. 同样添加 Ken Burns 动画（缩放 + 微旋转）
5. 关键：在 0:12.500（新月素材）处，让素材停留稍长（18 帧），作为视觉锚点
6. 调色层 "Grade_Part2"：
     色温向中性过渡（相比 Part1 减少 +10）
     清晰度 +10
```

### 步骤 5：制作第三阶段（三月升空）

```
1. 新建合成 "Part3_Ascension"
2. 背景图层：纯色 #0A0E27（深空蓝）
3. 导入新月 PNG，放置于画面中心偏上
4. 添加 Glow 效果：
     Glow Threshold: 60%
     Glow Radius: 25
     Glow Intensity: 1.5
     Glow Colors: A & B Colors（A: #E8F4F8, B: #4A90E2）

5. 使用钢笔工具绘制「3」形路径（与新月的弧度呼应）
   命名为 "Path_3"

6. 粒子实现（Particular）：
   - 新建纯色图层 → Particular
   - Emitter: Layer，选择 "Path_3"
   - Particles/sec: 150
   - Velocity: 80（Y轴负方向为主）
   - Size over Life: 从小变大再消失
   - Color: #E8F4F8 → #4A90E2
   - 添加 Turbulence Field 让轨迹有随机飘动感

7. 景深分层：
   - 复制粒子层 3 份
   - 远景：Blur 12px，Scale 50%
   - 中景：Blur 4px，Scale 85%
   - 近景：无 Blur，Scale 120%，添加 Motion Blur

8. 汇聚动画：
   - 在 0:18.000，添加空对象 "Attractor"
   - Particular 的 Physics → Air → Air Resistance 降低，Wind Y 设为负值
   - 或手动关键帧：粒子从分散状态被「吸」回新月中心
```

### 步骤 6：制作品牌卡

```
1. 新建合成 "Part4_Logo"
2. 背景：Ramp（径向渐变）从 #0A0E27 到 #000000
3. 添加微弱星图纹理（不透明度 8%，混合模式 Screen）
4. 文字层 "Kimi K3"：
   - 字体：Canela / Freight / 思源宋体 Heavy
   - 字号：100-120px
   - 颜色：#F0F0F0（非纯白，带微暖）
   - 字距：Tracking -20
5. 新月图标层：
   - 放置于文字上方 40px
   - 添加 Glow 和缩放脉动动画
6. 底部文字 "Moonshot AI / 月之暗面"：
   - 字号：18-20px
   - 颜色：#8A9AAF（ muted blue-gray）
   - 字距：Tracking +50（拉开，增加呼吸感）
```

### 步骤 7：最终合成

```
1. 新建合成 "Main_K3"
2. 按顺序排列：
   - Part1_Ancient（0:00 - 0:10）
   - Part2_Nature（0:10 - 0:16，与 Part1 交叉淡化 6 帧）
   - Part3_Ascension（0:16 - 0:19，叠加混合，从 Part2 的最后一帧淡入）
   - Part4_Logo（0:19 - 0:20.4）
3. 全局调整图层 "Final_Grade"：
   - Add Grain（数量 3%，大小 0.5）
   - 轻微暗角（Vignette，不透明度 15%）
4. 渲染输出：
   - 格式：H.264
   - 分辨率：720x720
   - 帧率：24fps
   - 码率：12-15 Mbps
```
