# Kimi K3 宣传片《三月纪》创意概念

> **核心概念**：「3」不是被发明的数字，而是月亮写在天上的签名，是人类探索文明中反复出现的轨迹。

---

## 一句话叙事

Kimi K3 不是凭空出现的科技产物，它是从三千年人类探索文明中自然升起的**第三个月相**。

## 符号策略

| 维度 | 设计 |
|------|------|
| 核心数字 | 3 |
| 视觉同构 | 新月负形 ≈ 数字 3 的上半弧线 |
| 文化根脉 | 东方古籍美学 + 太空探索影像 |
| 色彩弧线 | 暖宣纸黄 → 深空靛蓝（单镜头内完成色温跃迁） |
| 动态收尾 | 光点升空、星图连线、新月凝聚 |

## 四段结构

1. **古籍中的三**（0:00 – 0:10）：从《道德经》《三才图会》、三足鼎、敦煌星图等古籍/历史图像中，建立「3」作为文明密码的古老身份。
2. **万物中的三**（0:10 – 0:16）：从「人类书写的 3」过渡到「自然界本身就有 3」。三片茶叶、三枚铜钱、三潭印月、新月、DNA、三环星云……每张图本身就是 3。
3. **三月升空**（0:16 – 0:19）：静物活化。新月发光，光点画出「3」形轨迹，飞升成星图/神经网络，最终凝聚成新月。
4. **品牌卡**（0:19 – 0:20.4）：深空蓝背景，「Kimi K3」与发光新月居中，底部「Moonshot AI」淡入。

## 视觉风格

- **整体调性**：东方古籍美学 + 太空探索影像
- **色彩弧线**：#E8DCC8 暖宣纸黄 → #0A0E27 深空靛蓝
- **质感协议**：宣纸纹理、竹简边缘、灰尘划痕、胶片颗粒、轻微暗角
- **字体**：英文 Canela / Freight Display；中文思源宋体 Heavy / 方正清刻本悦宋

## 参考基准

借鉴 Anthropic Claude「Fable 5」宣传片的运动图形语法（快速蒙太奇、符号构成、粒子活化、品牌卡收尾），但进行彻底的**东方化 + 品牌符号化**改造。
