# 《三月纪》BGM 配乐脚本

> 版本：v6 对应版  
> 总片长：20.7s  
> 风格：新古典极简钢琴（Ludovico Einaudi / Philip Glass / 坂本龙一）  
> 情绪：克制、留白、从纸张到星空、东方理性、不煽情  

---

## 一、总体策略

### 1.1 乐器

- **仅一架独奏钢琴**（推荐 Yamaha CFX 或 Steinway D 采样）
- 禁止弦乐、电子铺底、pad、打击乐
- 所有延音/空间感仅通过钢琴自然延音与混响实现

### 1.2 两个版本

| 版本 | 用途 | 混响特征 |
|------|------|----------|
| **干声版（Dry / Intimate）** | 前段私密感，突出纸张、古籍、近景 | 小房间混响，短 decay（约 0.8–1.2s），前段 dominant |
| **湿声版（Wet / Cathedral）** | 后段星空感，强调升华、宏大 | 大教堂混响，长 decay（约 3.5–4.5s），后段逐渐 dominant |

最终混音：两版不是简单开关，而是**湿声从 0% 渐强到 100%** 的交叉淡化。前 8 秒以干声为主，第 8 秒开始湿声逐步进入，第 14 秒后湿声主导。

### 1.3 调性 / 音阶

- **主调：D 小调**
- 旋律以 D 自然小调 + D 五声音阶混合为主
- 避免西方古典的强烈解决感，多用悬停、延伸九音、保留音

### 1.4 速度

- **自由速度（Rubato）**，无固定节拍器
- 整体在 **♩ = 56–66** 之间浮动
- 前段偏慢（56），后段略流动（66），但始终保持克制

### 1.5 织体原则

- **少即是多**：单音、双音、三音和弦为主
- 左手尽量留白，多用低音单音或开放五度
- 右手旋律线条简短，像呼吸一样断句
- 大量休止符，让画面本身成为音乐的一部分

## 二、分镜配乐脚本

### 0:00 – 0:01.5｜引子：月牙初现

**画面**：黑色月球，一弯月牙光弧刚刚亮起。

**音乐指示**：
- 从完全的静默开始
- 在约 0:03 处，钢琴轻轻进入（不是 0:00，要给观众半秒喘息）
- 一个极低的 **D2 单音**，ppp，带踏板
- 随后 **A2** 五度呼应
- 最后 **D3** 像远处钟声一样轻轻落下
- 整个引子只有 3 个音，持续约 1.2s

**情绪**：像打开一本古籍前的屏息。

**技术**：干声版 dominant；湿声版只给 10% 混入，保持亲密感。

---

### 0:01.5 – 0:07.5｜A 段：纸张上的三（Phase 1）

**画面**：古籍、三足鼎、星图、手稿快速但缓慢地切换。

**音乐指示**：
- 建立在 **D 小调** 上的极简主题
- 右手主旋律（高音区）：
  ```
  D4 – F4 – A4 – G4 – F4 – D4
  ```
- 左手（低音区）：每 2 拍一个低音 D2 或 A1，像心跳
- 节奏稀疏：每小节 2–3 个和弦/音
- 使用大量踏板，让音符彼此渗透，产生"纸张泛黄"的朦胧感

**变奏逻辑**：
- 第 1 遍：单音旋律 + 低音 D
- 第 2 遍：加入 A4 双音，和声稍厚
- 第 3 遍：旋律结尾加入 E4（九音），制造悬停感

**情绪**：克制、审视、像博物馆里的阅读。

**技术**：
- 干声版 80% / 湿声版 20%
- 踏板长但轻，避免浑浊

---

### 0:07.5 – 0:10.5｜B 段启动：万物苏醒（Phase 2 前半）

**画面**：茶叶、铜钱、三潭印月、三角洲、三叶草……自然中的 3 开始出现。

**音乐指示**：
- 引入 **Philip Glass 式重复音型**
- 右手固定音型：
  ```
  | D4 – F4 – A4 | F4 – A4 – D5 | A4 – D5 – F5 |
  ```
- 左手以八分音符重复 **D2 – A2** 五度，像脉搏开始加速
- 速度从 ♩=56 逐渐提到 ♩=62
- 音域向中高音区扩展

**情绪**：从"观看"转为"发现"，开始有了一丝涌动。

**技术**：
- 干声版 60% / 湿声版 40%
- 湿声开始明显混入，空间感逐渐打开

---

### 0:10.5 – 0:14.1｜B 段加速：从自然到宇宙（Phase 2 后半）

**画面**：DNA、星云、火箭蓝图、卫星轨道、光纤……3 出现得越来越快。

**音乐指示**：
- 重复音型继续，但加入变化：
  - 右手在高音区加入 **A5** 闪烁音
  - 左手从五度变为 **D2 – F2 – A2** 完整小三和弦分解
- 每 2 小节升高一个八度，制造"向上"的升腾感
- 在第 0:13.5 左右，达到一个小高峰：
  ```
  右手：D5 – F5 – A5 – D6（琶音上行）
  左手：D2 持续低音
  ```
- 但**不要弹满**，每个高峰后立即留白 0.5s

**情绪**：克制中的张力，像水面下的暗涌。

**技术**：
- 干声版 30% / 湿声版 70%
- 混响 decay 开始变长，空间从"房间"过渡到"大厅"

---

### 0:14.1 – 0:17.6｜C 段：三月升空（Phase 3）

**画面**：月光画出 3 的轨迹，光点飞升，连成星图，凝聚成新月。

**音乐指示**：
- **全曲最高点**，但音不要多
- 右手高音区：一个拉长的 **D6**，像星光
- 左手低音区：D2 持续音 + A2 泛音
- 中间加入稀疏的 **F5 – A5 – D6** 琶音，模拟光点上升
- 整体织体：
  ```
  低音 D2（踏板全踩）
  中音 F3 – A3（每拍一次，像心跳）
  高音 D6 – F6 – A6（闪烁，不规则节奏）
  ```
- 在 0:16.5 左右达到一个**不解决的高潮**，然后突然收住

**情绪**：从纸张彻底进入星空，克制但辽阔。

**技术**：
- 干声版 10% / 湿声版 90%
- 大教堂混响全开，decay 3.5–4.5s
- 踏板保持，让和声在空间中延续

---

### 0:17.6 – 0:20.0｜D 段：品牌卡（Kimi K3 + Moonshot AI）

**画面**：深空背景，发光新月，品牌文字淡入。

**音乐指示**：
- 从 C 段高潮突然回落
- 回到 **D4 – A4 五度**，像 A 段的回声
- 节奏极慢，每个音之间留 0.8–1.0s 空白
- 在 0:18.8（Moonshot 卡出现）时，加入一个极轻的 **F4**，作为和声色彩
- 最后 3 个音：
  ```
  D4（Kimi 卡出现）→ F4（Moonshot 卡出现）→ A4（文字定格）
  ```

**情绪**：聚焦、神秘、品牌出现时的庄重。

**技术**：
- 湿声 dominant，但音量轻
- 混响长 but sparse，让每个音像星光一样慢慢消散

---

### 0:20.0 – 0:21.5｜尾声：新月定格

**画面**：收尾月球光环，月牙光弧定格，视频结束。

**音乐指示**：
- 一个极轻的 **D2 泛音**（或 D3）
- 随后完全静默 0.5s
- 最后一个 **A1**，ppp，让低音在混响中慢慢消失
- 总音乐余韵持续约 3–4s（靠混响自然衰减）

**情绪**：余韵、留白、像合上书页后的沉默。

**技术**：
- 湿声版 100%，大教堂混响
- 干声版 0%
- 最后不要硬切，让自然衰减结束

## 三、两个版本的制作规格

### 3.1 干声版（Dry / Intimate）

| 参数 | 规格 |
|------|------|
| 混响类型 | 小型木质房间 / Studio |
| Reverb Time | 0.8–1.2s |
| Pre-delay | 10–15ms |
| 高频阻尼 | 中等 |
| 适用段落 | 0:00 – 0:08 |
| 情绪 | 私密、纸张、近景 |

### 3.2 湿声版（Wet / Cathedral）

| 参数 | 规格 |
|------|------|
| 混响类型 | 大教堂 / Concert Hall |
| Reverb Time | 3.5–4.5s |
| Pre-delay | 25–40ms |
| 高频阻尼 | 轻，保留空气感 |
| 适用段落 | 0:14 – 0:21.5 |
| 情绪 | 星空、升华、宏大 |

### 3.3 最终混音交叉淡化

```
时间轴：
0:00  ───── 干声 100% ─────┬──── 干声 70% / 湿声 30% ────┬──── 干声 30% / 湿声 70% ────┬── 湿声 100% ──→
                          0:08                        0:12                        0:16
```

## 四、分轨交付要求

### 4.1 分轨清单

| 分轨名 | 内容 | 声像 |
|--------|------|------|
| `PIANO_LEFT.wav` | 左手低音部分（D2–A3） | 左 30% |
| `PIANO_RIGHT.wav` | 右手高音部分（D4–A6） | 右 30% |
| `PIANO_DRY_STEM.wav` | 干声版立体声总线 | 中央 |
| `PIANO_WET_STEM.wav` | 湿声版立体声总线 | 中央 |
| `PIANO_MIX.wav` | 最终混音（干→湿交叉淡化） | 中央 |

### 4.2 技术规格

- **采样率**：48kHz
- **位深**：24bit
- **格式**：WAV（ Broadcast Wave ）
- **峰值电平**：-3dBFS 以下
- **动态范围**：保留自然动态，不要过度压缩

---

## 五、演奏提示

1. **触键**：前段用指腹慢触，后段指尖清晰但不过亮
2. **踏板**：前段长踏板 but 轻踩，后段半踏板换得更频繁，避免浑浊
3. **动态范围**：整体 ppp 到 mf，**不要出现 f**
4. **留白**：每个乐句后必须留白，留白本身就是音乐
5. **东方理性**：避免浪漫主义的 rubato 起伏，速度变化要像呼吸一样自然
